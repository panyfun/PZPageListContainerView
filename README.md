# PZPageListContainerView


## #特性

PZPageListContainerView是一个能够支持上下左右滑入(类似SnapChat)或纵向/水平滑动(类似头条)的PageList基础框架。

它能够实现这样的框架效果

* 类似SnapChat首页的多方向滑入

  ![SnapChat风格](http://img.pany.fun/18-3-28/27022081.jpg)

* 类似头条的横向多页面滑动

  ![头条风格](http://img.pany.fun/18-3-28/85028526.jpg)
